九月 3, 2024, 11:49 中午

Vue 從模板傳值到 CSS 偽元素 content 內容

使用 Attr()

首先在模板定義data-*參數，並使用 Vue 模板傳入對應值

<div class="dh-header" :data-title="data.title"></div>

然後在 CSS 中使用attr()引入自訂的變數

.dh-header:before {
    content:attr(data-title);
    display:block;
}

什麼是 HTML 5 中的資料屬性（data-* attribute）
