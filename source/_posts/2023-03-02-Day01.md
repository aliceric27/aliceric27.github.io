### [Running Sum of 1d Array - LeetCode](https://leetcode.com/problems/running-sum-of-1d-array/?envType=study-plan&id=level-1)

主要求每個數值為前一個數值的加總，我是用比較暴力的解法，直接加總後 push 到結果陣列後輸出

```js
var runningSum = function(nums) {
    let r = []
    let sum=0;
    for(let i=0;i<nums.length;i++){
        sum+=Number(nums[i])
        r.push(sum)
    }
    return r
};
```

最佳解法

使用 `for(...of...)`，這個內建迴圈函數可以循序遍歷陣列內所有的值

[MDN](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Statements/for...of)教學範例

```js
let iterable = [10, 20, 30];

for (let value of iterable) {
    value += 1;
    console.log(value);
}
// 11
// 21
// 31
```

實作

```js
var runningSum = function(nums) {
    let r = []
    let sum=0;
    for(let i of nums){
        sum+=i
        r.push(sum)
    }
    return r
};
```

### [Find Pivot Index - LeetCode](https://leetcode.com/problems/find-pivot-index/)

這題一開始還看不太懂，總之就是要求 arr[n] 左邊的數值總和和右邊數值總和是否相等，這個數值不包含 arr[n] 本身，

如果 arr[0] 則左邊沒有數值則左邊數值總和則為 0

優先尋找到 arr[n]

arr[n] 的左邊數值總和右邊數值總和相等，則 return  n

arr[n] 的左邊數值和右邊數值找到最後 (arr[arr.length-1]) 都不相等的話則 return -1

如果 arr[n] 的左邊與右邊數值都為 0 則 return 0

我的解法，暴力找出所有組合一一比對

```js
var pivotIndex = function(nums) {
let n = 0
while(n<nums.length){
    let left =0
    let right =0
    if(n===0){
        for(let i=nums.length-1;i>n;i--){
            right+=nums[i]
        }
        if(right===0) return 0
    }else{
        for(let i=0;i<n;i++){
            left+=nums[i]
        }
        for(let i=n+1;i<nums.length;i++){
            right+=nums[i]
        }
        if(left===right){
            return n
        }
    }
n++
}
return -1
};
```

最佳解法，這個解法參考老外的，非常巧妙

先將所有陣列值都先加總為右邊的總和，然後再迴圈慢慢扣除每個數值，扣除掉一個數值如果不相等就加到左邊的變數中，非常棒也非常省時的解法!

```js
const pivotIndex = (nums) => {
 // We have not started our program yet, so values are at 0
    let rightSum = 0;
    let leftSum = 0;

 // Calculate the sum of the entire array
    // O(n)
    nums.forEach((v) => (rightSum += v));

    // O(n)
    for (var i = 0; i <= nums.length - 1; i++) {
        let curr = nums[i];
  
  // The right sum no longer contains the curr number
        rightSum -= curr;
  
  // if left sum equals right sum, we return index
        if (leftSum === rightSum) return i;
  
  // We did not find a match
  // Left sum now contains our curr number
        leftSum += curr;
    }
    return -1;
};
```
