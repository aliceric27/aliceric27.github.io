## 為什麼離職

我實在很受不了需求無限增加之術，我很好配合不代表什麼需求都可以隨便丟過來欸，在原公司測試還會丟一些奇妙的需求，印象比較深刻的有一項

QA:那個閒置的判斷可以不要讓滾輪也可以觸發嗎?

:什麼意思?

QA:就是用戶在頁面上下滾動，有可能他不在頁面上使用，不應該判斷不是閒置狀態

OS:(誰會沒事在頁面上滾來滾去的，而且滾來滾去用戶沒有使用，不就代表是用腳本?誰沒事會用腳本在頁面上掛腳本讓他滾來滾去)

...



因為這種很莫名其妙的需求要花很多力氣去說服 QA 不要這樣，浪費了很多溝通成本

其次是很多需求規劃的經驗不足，因為有RWD預期的規劃，我認為當初應該要事先實作手機版才是正確的，當然也是因為我的經驗不足，我當初以為這個網站只是一個簡單的形象網站吧?

結果是一個遊戲的官網，還要完全由我這個新人規劃和實作，想當然爾下場非常慘烈。

各種套件混用、寫法放飛(沒有 Codereview)..等問題

當然最終畫面和功能是可以顯示的，不過整個專案是完全沒有維護性的，因為根本沒有人告訴我這樣做是不是正確的。

最終我認為在這邊是無法有效學習到更多專案上實務才會選擇離開的，更多的只有怎麼應付需求。

當然也是在這邊有學習到比較有用的東西像Vue框架、Typescript、API串接、SCSS框架等

不過讓我學習到更多的是不管會多少技術，都沒辦法幫助我處理對人的問題，這個問題我認為是一間公司的風氣吧，至少我對於沒有加班費這事是比較敏感的，最後才會選擇離職。

## 面試心得

{% note info%}公司名打碼{% endnote %}

彙整我這一個禮拜面試的一些公司心得

###  A某博弈公司 - 已錄取未報到

無筆試、2位主管面試

主要面試的有技術主管和另一位看似主管職的人(?)，簡單自我介紹後詢問上一份公司專案使用到的相關技術，然後詢問期望薪資，過程有提到主要分工會有靜態畫面製作的人員和功能製作的前端小組，我入職後會加入功能製作的這個 team ，比較不用顧及前端切版

有拿到  offer 約43K+2K，原本因為這間公司錄取要去報到，無奈那時候有通電話聯繫我報到時間，

那時回覆:我稍後會再回電，然後因為自己的疏忽導致忘記回電

再次致電時已被告知因為我沒有回電所以當作拒絕 offer 。

這是我的疏忽，結果也因為這次事件導致我接下來整個禮拜的面試之旅。

------

### B某博弈(?)公司 - 8/24面試至今無聲卡

無筆試、1位主管面試

這間公司印象很深刻的是大樓超破，很像我家裡鄉下的五金行的廁所那種感覺，然後辦公室很像逃難一樣不知道是剛搬遷過來還是怎樣，整個很亂，然後主管跟我說他們公司 PM 都是技術職出生，對於需求掌握很精準，還吹說專案都是先拿錢才開搞，主要是扁平化團隊，基本上就是一個專案一個前端，有提到專案有做過成人網站(?)，還說之前公司內鬥很嚴重才另外開一間新的公司，才會有這個職缺，整個就是一個很怪的公司，整個公司狀態和產品也很令人擔憂。

------

###  C某香港博弈公司 - 感謝函

無筆試、1主管+1HR面試

這間公司感覺滿有錢的，主要由技術主管面試，然後由 HR 進來說明薪資相關，不過 HR 有點急迫，感覺要趕快打發我走一樣，我後來思考了一下，應該是因為這間公司有比較多陳舊的專案是用 class 相關的 react 框架下去寫的，會需要這方面的經驗，我是回答會需要一點時間學吧，就涼涼了，不過找我去原本是詢問我是否願意學習 react nactive，可能後來有熟悉 react nactive 的人去應徵了吧，看到面試趣有人是很熟悉 class 寫法就錄取的，所以我就被刷掉了。

------

### D某 IC 相關研發公司 - 9/1面試 9/8二面

無筆試，3~4位主管(有主管面試途中加入)

簡單自我介紹後就就詢問我相關作品的的使用技術，然後有詢問到一些關於該公司專案之後使用的相關技術是如何達成與實現，問的都是我剛好上一份工作有碰到的技術，所以回答滿簡單的，也有給我看該公司內部專案的樣子，在我看來跟我之前公司專案比起來簡直是一塊蛋糕，整個頁面是屬於形象網站類型的，也已經有外包的人完成整個架構，應該是很好上手才對，過程比較融洽的，加上研發的 IC 相關是屬於音頻相關，最近剛好再玩喇叭所以還算有點興致，缺點可能是入職後只有我一個前端，等於我又是公司前端扛霸子了，這個招牌壓力真的有點大阿QQ。

9/8有去第二次面試，面是我的是一位人事主管(?)，感覺大公司在面試上會比較瑣碎一點，因為面試後這位人事主管還要回去和總經理報告討論之類，也說這間公司未來會爭取上市上櫃，覺得是一間滿有淺力的公司，並表示如果有入職會優先選擇貴公司。

------

### E某全遠端博弈公司 - 9/4面試至今無聲卡

無筆試、3位線上面試

簡單自我介紹後就詢問我作品相關技術，然後也有提到上一份專案的 code ，可能想要看我上一份專案是怎麼完成的，因為那時沒有備份，所以沒有開，最後開的是剛學習所完成的專案內容詢問，有問到 promise 相關問題我回答錯了，然後還有詢問一些簡易問題，最後有提到公司是全遠端當如果想去公司也有**客服辦公室**可以用聽到關鍵字後就知道為什麼會需要全遠端了，感覺就是風險有點高的公司，謝謝在聯絡耶。

------

### F某接案公司 - 9/5 面試至今無聲卡

無筆試、HR線上面試

比較有意思的是這間公司 HR 會問星座相關問題，幸好我有備題，也是滿傻眼為什麼會問這個問題，最後有提到薪資福利相關，覺得這間薪水應該是面試過偏低的，除非走頭無路不然這間優先度應該會排比較後面。

------

### G某民營企業子公司 - 9/6 面試至今無聲卡

有筆試、2主管分別面試

主要題目是 React 相關，辦公室舊舊的，技術主管主要針對我之前作品和他們的系統做了一個滿詳盡的介紹，第二位主管算是更上層的主管，主要介紹薪資福利相關，薪資比我想像的還要多，僅次於 D 公司，感覺也是不錯的缺，前端也是有多位同事，應該可以學習到滿多東西的，技術主管感覺也滿有料的，還有上市櫃民營企業撐腰，排除辦公室有點舊舊不算新創，應該也是很穩定不錯的肥缺。

------

## 結語
