## 聲明變數

TS 基本的聲明變數方式：

* 在變數後加上冒號 `let a:number`即為將 a 聲明為 number，無法賦值 number 外的數值

```typescript
let a:number;   //聲明 a 為 number
a=10;
a=30;
a='hello';   //報錯，因為 a 變數已經聲明為number

//通常都用以下寫法
let c:boolean = falsel; 
//如果變數的聲明和賦值是同時進行的，TS可以自動對變數檢測類型;
let c = false ;  //c = boolean
```

* 函數聲明

```typescript
//在 function 中聲明參數類型
function sum( a:number, b:number):number{  //如果要聲明返回值參數類型則在判斷後加上聲明
 return a+b;
}

sum(a:123,b:'456');  //TS會報錯
```

## 額外的類型補充

* 字面常數(字面量)：類似 const 賦值，當指定常數後，變數則無法賦值指定常數外的值

```typescript
let a:10;
a=10; //正常
a=20; //錯誤

//可以使用 | 來連接多個字串或多個類型
let b: "boy" | "girl" //b可以賦值為字串 'boy'或'girl'
let c: boolean | number //c可以賦值為布林值或數字

```

* any 表示任意值，盡量少用這個類型

```typescript
//any 表示任意類型，聲明變數為any當作關閉ts的類型檢測
//使用TS時不建議使用此類型
//如果不指定類型，則TS會自動將變數判斷類型為any(隱式的any)
let d: any;
let d2;  //隱式的any
```

* unknow 為安全版 any

```typescript
//表示未知類型的值;
let e :unknown;
let s:string;

e='hello';
//unknow 實際上是一個安全的any
//unknow 變數無法直接賦值給其他變數
if(typeof e ==='string'){
    s=e;
}
```

* as 斷言類型，強制告訴解析器使用什麼類型來對變數解析
語法：`變數 as 類型`、`<類型>變數`

```typescript
//AS類型斷言，可用來告訴解析器變數的實際類型
/*

語法：
變數 as 類型
<類型>變數

*/
s = e as string;
S=<string>e;

```

* void 與 never
void 表示為回傳空值，never 表示永不回傳值，通常用於除錯

```typescript
//void 表示空值，以函數為例，表示沒有返回值
function fn():void{
}
//never 表示永遠不會回傳，通常用來回報錯誤
function fn2():never{
    throw new Error('錯誤');
}
```

## 物件相關類型

* 物件相關類型宣告方式：

1. 通常使用`{}`來指定物件可以包含哪些屬性
2. 語法：`{屬姓名:屬性值,屬性名:屬性值}`
3. 在屬性名後面加上?，表示屬性可選(賦值時該屬性可有可無)，沒加則為必需值

```typescript
// object 表示一個js物件
let a:object;
a = function(){
}

//{}用來指定物件可以包含哪些屬性
//語法:{屬姓名:屬性值,屬性名:屬性值}
//在屬性名後面加上?，表示屬性可選(可有可無)，沒加則為必需

let b :{name:string,age?:number};

b={name:'胖胖胖',age:19};

//[propName:string]:any 表示任意類型的屬性
let c :{name:string,[propName:string]:any};
c={name:'莓莓莓',age:18,gender:'男'}
```

* 設置結構函數的類型聲明：
強制該變數需要按照特定規則傳入指定參數與指定類型，返回值也要按照設定返回

使用語法：`(參數:類型,參數:類型...)=>返回值(類型)`

```typescript
let d: (a:number,b:number)=>number;

// 錯誤寫法
/* d=function(n1:string,n2:string):number{
    return 10;
 } */
 
```

* 陣列的類型聲明
使用語法：`類型[]`、`Array<類型>`

```typescript
// string[]表示字串陣列
let e:string[];

//number[]表示數值的陣列
let f:number[];

let g:Array<number>;
g=[1,2,3];
```

* 元組的類型聲明
元組表示固定長度的陣列
語法：`[類型,類型,類型]`

```typescript
let h:[string,number]

h=['hello',123];    //一定要與宣告的陣列長度與類型一致

```

* enum 列舉(枚舉)類型
將所有可能情況預先設定好，並直接使用

```typescript

enum Gender{
    Male,
    Female
}

let i : {name:string,gender:Gender};
i={
    name:'小胖胖',
    gender:Gender.Male
}

// console.log(i.gender===Gender.Male);

```

* & 表示與(同時)

```typescript
let j :{name:string}&{age:number};
// j={name:'大胖胖',age:20}; 
```

* 類型的別名
`type 自訂變數 = 聲明條件`
使用如果聲明一個條件判斷且要複用可以使用此宣告方式

```typescript
type myType = 1|2|3|4|5;
let k : myType ; 
let l : myType ; 
let m : myType;
k=2;
```
