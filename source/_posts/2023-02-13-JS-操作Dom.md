`document.createElement('element')` - 建立元素

Element`.append(element)` - 新增子元素

Element`.remove(element)` - 從 DOM 移除元素







``` html
<div id="app">
<h1>這是標題</h1>
  <p id="para">這是段落文字</p>
</div>
```



```js
var apple=document.createElement('h1');
//使用內建函數createElement來建立一個新元素
apple.textContnet="這是標題";
//修改元素屬性

var bee = document.getElementById('app');
bee.append(apple);
//使用append()函數，使目標於其指定項目的子元素中

alert(bee.children[0].textContent);//'這是標題'
alert(bee.children[1].textContent);//'這是段落文字'
//使用children[x]，可以選取第x個子元素

var para=document.getElementById('para');
alert(para.parentElement.textContent);
//使用parentElement，可以選取到父元素
```



學習怎麼抓到事件發生當下的元素

綁定函式後使用`event.target`來找到當前發生事件的元素

```html
<button onclick="hello()">蘋果</button>
<button onclick="hello()">香蕉</button>
<button onclick="hello()">西瓜</button>
```

```js
function hello()
{
  alert(event.target.textContent)
}
```
