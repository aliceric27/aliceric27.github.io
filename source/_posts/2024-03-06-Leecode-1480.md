Leecode 1480 Running Sum of 1D Array

<!--more-->

---


題目要求:輸入一個陣列，返回一個陣列
法1:兩個迴圈(o)n
法2:直接保存前一個總和，並修改原始陣列(o)1

```c#

public class Solution {
    public int[] RunningSum(int[] nums) {
            for (int i = 1; i < nums.Length; i++)
            {
                nums[i] +=nums[i - 1];
            }
            return nums;

}
}
```
