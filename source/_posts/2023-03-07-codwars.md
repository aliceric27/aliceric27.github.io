## [Greed is Good | Codewars](https://www.codewars.com/kata/5270d0d18625160ada0000e4)

給一個陣列看裡面的數字數量，如果1為3個1組+1000，以此類推，各數字3個1組都會加不同的分數，

如果數字不滿3位只有1和5會加分，剩下不加分。

首先必須知道陣列中1~6每個數字共出現幾次

這邊我用一個 a 的空陣列，並練習使用 .map 來跑迴圈計算，每個 dice 的 value 放入對應的 a 陣列 index中

並使用判斷，如果加到 3 就將結果計算一次，最後再算剩下的1 和 5結果加上就是答案了

雖然程式碼有點醜，不果我覺得這樣很好理解🤣

```js
function score( dice ) {
let a =[0,0,0,0,0,0,0]
let r = 0
dice.map((v,i)=>{
  a[v]++
  if(a[v]>=3){
    switch(v){
      case 1:{
        r+=1000
        a[1]=0
        break;
    }
      case 2:{
        r+=200
        a[2]=0
        break;
    }
        case 3:{
        r+=300
        a[3]=0
        break;
    }
        case 4:{
        r+=400
        a[4]=0
        break;
    }
        case 5:{
        r+=500
        a[5]=0
        break;
    }
        case 6:{
        r+=600
        a[6]=0
        break;
    }      
    }
  }
})
  r+=a[1]*100
  r+=a[5]*50
  return r
}
```
