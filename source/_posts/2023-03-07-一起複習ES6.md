## 變數宣告

[<img src="https://ithelp.ithome.com.tw/upload/images/20190916/20119924oZvDEyQZ98.png" alt="javascript 變數宣告 let、const、var 比較" style="zoom:33%;" />](https://ithelp.ithome.com.tw/articles/10213188)

分辨 `var`、`let`、`const`差異?

`var`：如果使用 `var`會導致變數洩漏到全域，導致域名汙染。

`let`：如果使用 `let`則作用域限定為此 代碼塊區域 {} 內。

`const`：無法被重新賦值。

## 模板字串符

[<img src="https://ithelp.ithome.com.tw/upload/images/20190904/20119924sqPihtpizr.png" alt="【ES6 小筆記】JS 組字串好方便！一目瞭然最清楚（模板字串符 template literal）" style="zoom:33%;" />](https://ithelp.ithome.com.tw/articles/10213611)

替代以往字串拼接方式，假設我想要在 Js 撰寫一些 Html 字串用於輸出，則以往寫法

```js
str ='<ul class="menu"><li><h3>' + name + '</h3><p>' + price + '元</p></li></ul>'
```

使用模板字串，連換行都可以一併帶入，其中模板字串也可以插入表達式

```js
str =`
<ul class="menu">
  <li>
    <h3>
      ${name}
    </h3>
    <p>${price}元</p>
    <p>${price * 0.9}元</p>     //表達式
  </li>
</ul>
`
```

## 箭頭函數

[<img src="https://ithelp.ithome.com.tw/upload/images/20190905/20119924v53BiHnbFY.png" alt="Day04【ES6 小筆記】箭頭函式 - 基礎使用範例（Arrow function ）" style="zoom:33%;" />](https://ithelp.ithome.com.tw/articles/10213501)

1. 寫法為 `() => {}`
2. 參數只有一個時，可以省略小括號 `()`
3. 省略大括號 `{}` 的話，代表直接 return 第一行程式碼
4. 如果返回值或參數有包含 `{}` ，需要在外圍加上括號 `()`
5. 如果有多行程式，需要加上大括號 `{}`

覺得第4點滿重要，因為使用 Vue 框架時常需要返回物件，會忘記加 `()`

## 參數預設值

[<img src="https://ithelp.ithome.com.tw/upload/images/20190905/20119924QqDPfQwN8o.png" alt="Day06【ES6 小筆記】函式參數預設值 - 基礎範例（Default Arguments）" style="zoom:33%;" />](https://ithelp.ithome.com.tw/articles/10213774)

在過去，設定函式預設的參數值時，需要這樣寫：

```js
function count(a,b) {
  a = a || 2;
  b = b || 3;
  return a + b;
};

console.log(count(6,3)); // 9
console.log(count()); // 5 因為沒有代入參數，所以會出現預設的a=2,b=3
```

而在 ES6 之後，提供了一個更簡單的方法：

```js
function count(x = 3, y = 2){
  console.log(x+y);
}
count(); // 5
```

如果省略帶入參數的話，丟入的參數值會 `由前往後`代入

```js
//只帶入一個參數
function count(x = 3, y = 2){
  console.log(x+y);
}
count(5); // 7

//只有第一個參數有預設值
function count(x = 3, y){
  console.log(x+y);
}
count(5); // Nan  x=5 , y= undifined
count(2,1); // 3  x=2 ,y=1

//只有第二個參數有預設值
function count(x, y=3){
  console.log(x+y);
}
count(2); // 5  x=2 , y=3
count(2,6); // 8  x=2, y=6
```

如果傳入的值為undefined，則會出現預設值

```js
function count(x = 3, y){
  console.log(x+y);
}
count(undefined,2); // 5
```

箭頭函式的寫法

```js
let count = (x = 3, y = 2) => console.log(x+y) ;
count(); // 5
```

## 剩餘參數

[<img src="https://ithelp.ithome.com.tw/upload/images/20190909/20119924YtB40epAqD.png" alt="【ES6 小筆記】剩餘參數 - 基礎使用範例（Rest parameter）" style="zoom:33%;" />](https://ithelp.ithome.com.tw/articles/10213774)

特性

- 取得剩下的參數，並存為一個**陣列**。
- 剩餘參數在傳入時，必定是**位於最後一位**，並且只能有一個剩餘參數。
- 剩餘參數的值在沒有傳入實際值時，會變為一個**空陣列**，而不是 `undefined`。
- 又叫做「剩餘運算子」「其餘參數」「其餘運算子」
- ESLint（airbnb）規則：不要使用 `arguments` 語法，使用 rest 語法 `...` 替代

基本使用方式

函式中，最後一個參數命名開頭使用 `...` 當開頭，就會將**當前位置~最後一個**變數組成陣列

```JS
function f(a, b, ...變數名稱) {
  //...
}
```

將所有參數轉成陣列

範例，我有一組套餐set，將套餐轉成陣列

```js
function set(...food){
console.log(food) // ['大麥克','辣脆雞','冰炫風','蘋果派']
}
set('大麥克','辣脆雞','冰炫風','蘋果派')
```

取完特定變數後的剩下資料

```js
function set(bigmac,spicy,...food){
console.log(bigmac) //'大麥克'
console.log(spicy) //'辣脆雞'
console.log(food) //['冰炫風','蘋果派']
}
set('大麥克','辣脆雞','冰炫風','蘋果派')
```

也可以搭配 `.map()`混和使用做運算

```js
function set(...food){
console.log(food.map(Discount))
function Discount(arr){
return Math.floor(arr*0.85) // [170, 85, 42, 25]
}
}
set(200,100,50,30)
```

## 展開運算子

[<img src="https://ithelp.ithome.com.tw/upload/images/20190904/20119924sqPihtpizr.png" alt="【ES6 小筆記】JS 組字串好方便！一目瞭然最清楚（模板字串符 template literal）" style="zoom:33%;" />](https://ithelp.ithome.com.tw/articles/10213611)

與剩餘運算子用法相同都是 `...`，差別在展開運算可以將陣列內取值。

**基本用法**：

用在陣列中會將陣列拆開，並用逗號隔開，用在字串也會將每個字符拆開

```js
let day=[1,2,3,4,5,6]
console.log(...day)
let str='gAyNotGay'
console.log(...str)
```

**用來複製陣列**：

```js
let set = ['大麥克','大冰奶']
let copy = [...set];
console.log(copy) //['大麥克','大冰奶']
```

**用來組合兩個陣列：**

ES6以前使用 `.concat(arr)`方法來組合兩陣列

```js
let food = ['大麥克','麥香魚']
let drink= ['大可','大冰奶']
let fat = food.concat(drink);
console.log(fat) //['大麥克', '麥香魚', '大可', '大冰奶']
```

使用 `...`合併，達到同樣效果

```js
let food = ['大麥克','麥香魚']
let drink= ['大可','大冰奶']
let fat = [...food,...drink];
console.log(fat)//['大麥克', '麥香魚', '大可', '大冰奶']
```

**用來把陣列傳入函式做計算：**

```js
function count(a){
   return a+100
}
let price =[100,200,300,400];
console.log(count(...price)) //200  因為只帶入一個參數
------------------------------------------------------------------------------
function count2(a,b,c,d){
   return a+100+b+c+d
}
let price =[100,200,300,400];
console.log(count2(...price)) //1100  帶入四個參數
------------------------------------------------------------------------------
```

## 物件的解構賦值

[<img src="https://ithelp.ithome.com.tw/upload/images/20190911/20119924tfYqYv7Adk.png" alt="物件的解構賦值-以一間好吃的餐廳為例）" style="zoom:33%;" />](https://ithelp.ithome.com.tw/articles/10214721)

### 基本使用

假設我們有一個班級物件

```js
let myclass = {
    name:"A班",
    teacher:"恰北北",
    studet:{
        name:"胖胖",
        number:"13011",
        weight:"100kg"
    }
}
```

如果要提取出`name`、與`teacher`

```js
//過往寫法
let name=myclass.name
let teacher=myclass.teacher

//ES6賦值解構寫法
let {name,teacher} = myclass
```

這樣結果都是`name="A班"`與`teacher="恰北北"`

### 重新賦予變數的名稱

變數名稱預設會是物件的屬性名稱，但如果不喜歡的話，可以重新賦予變數新的名稱，例如以下我們把屬性 teacher 的值取出來，重新命名為 dog，就可以這樣寫：

```js
let{teacher:dog}=myclass
//dog="恰北北"
```

如果想要取得 student 內的屬性可以這樣寫：

```js
let {name,number,weight}=myclass.studet
//name="胖胖",
//number="13011",
//weight="100kg"
```

也可以取出值後重新賦值

{% note info%}
重新賦值後，原本的值則不會被宣告
讀取該變數會拋出錯誤`ReferenceError: name is not defined`
{% endnote %}

```js
let {name:fat,number:no,weight:kg}=myclass.studet
//fat="胖胖",
//no="13011",
//kg="100kg"
```

### 設定預設值

假設物件 myclass 裡沒有 tall，我們卻在宣告時設定 tall 的話，tall 就會出現 undefined

```js
let {name,number,weight,tall}=myclass.studet
//tall=undefined
```

如果不想出現 undefined，我們可以在宣告同時給變數一個預設值，寫法如下：

```js
let {name,number,weight,tall="180cm"}=myclass.studet
//tall="180cm"
```

### 函數的應用

解構也能使用在函式的參數，使用方式如同將傳入的物件對應到函式參數上，以下範例簡單示範自訂變數名稱、順序、預設值的各種情況：

```js
let love = ({a='胖胖',b="甜點"})=>{
console.log(`${a} 愛 ${b}`)
}
love({})//胖胖 愛 甜點
love('我', '睡覺');//胖胖 愛 甜點
love({a:'我',b:'睡覺'})//我 愛 睡覺
love({b:'工作'});//胖胖 愛 工作
love({a:"莓莓",b:"我"});//莓莓 愛 我
```

## 陣列的解構賦值

[<img src="https://ithelp.ithome.com.tw/upload/images/20190911/20119924FyNJwriFYE.png" alt="陣列的解構賦值-以哥哥的前女友為例" style="zoom:33%;" />](https://ithelp.ithome.com.tw/articles/10214766)

### 陣列解構特性

陣列的解構賦值是使用`順序的索引值`對應，就像是鏡子一樣，將右方的資料往左邊對應，然後會一個位置對一個值 (但沒有像鏡子左右顛倒)，如圖：

![圖 6](https://i.imgur.com/dvtqs5x.png)

像這個樣子 我們就可以一次宣告好全部的變數，得到 a=1,b=2,c=3,d=4,e=5，以下來看看範例囉～

### 基礎範例

假設有一個陣列紀錄昨天吃到飽的食物

```js
let food = ['龍蝦','炸雞','牛排','鮑魚'];
```

如果我們要把每個食物的名稱抓出來存在變數，一般會需要透過陣列的索引值，才能取得我們要的值並賦予變數，像這樣：

```js
let one = food[0];
let two = food[1];
let three = food[2];
let four = food[3];
```

而在陣列解構賦值就可以這樣寫：

```js
let [one, two, three, four] = food;
//one=龍蝦,two=炸雞,three=牛排,four=鮑魚
```

如果遇到空的變數，這些值也會跳過（因為是對應索引位置）。

```js
let exGirl = ['蘋蘋','欣欣','雯雯','綺綺'];
let [one, , three, four] = exGirl;
// one='蘋蘋'
// three='雯雯'
// fore='綺綺'
```

### 剩餘參數&陣列解構的運用

如果想知道第二個值以後的剩餘值有哪些，可以搭配之前介紹過的[剩餘參數](#剩餘參數)這樣寫：

```js
let exGirl = ['蘋蘋','欣欣','雯雯','綺綺'];
let[one,two,...others]=exGirl
// one='蘋蘋',two='欣欣'
//others=['雯雯','綺綺']
```

### 預設值

可以在陣列解構內宣告新的值

```js
let exGirl = ['蘋蘋','欣欣','雯雯','綺綺'];
let [one, two , three , four , five ="月月"] = exGirl;
console.log(five) // five = 月月
```

### 變數交換

在陣列解構中做互換變數非常的方便，如果發現第一位數與第二位數寫反要交換，就可以這樣寫：

```js
let one = '蘋蘋';
let two = '欣欣';
[one, two] = [two, one];
console.log(one); // one = 欣欣
console.log(two); // two = 蘋蘋
```

### 字串拆解

如果陣列解構遇到字串，則會將字串拆解成一個一個字元，賦予到左方的變數上。

```js
let please = '梓喵買醬油';
[a, b, c, d,e] = please;
// a=梓,b=喵,c=買,d=醬,e=油
```
