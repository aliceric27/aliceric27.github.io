### [Array.prototype.map()](https://developer.mozilla.org/zh-TW/docs/Web/JavaScript/Reference/Global_Objects/Array/map)

`map()` 方法會建立一個新的陣列，其內容為原陣列的每一個元素經由回呼函式運算後所回傳的結果之集合。

所以 `map()` 可以帶入 function 並回傳一個新的陣列

`map()` 實際使用範例

```js
const arr = [1,2,3,4,5]; //宣告一個陣列
function double(a){		//設定一個 function 將傳入的參數*2並回傳
    return a*2  
}
console.log(arr.map(double)); //印出[2,4,6,8,10]

```

實做仿照基礎`map()`輸出

```js
function fakemap(arr,callback){     //設定一個 function 並傳入陣列與 function
    let result = [];			//result 存放產生的新陣列
    for(let i=0;i<arr.length;i++){		//使用迴圈將所有 arry 值遍歷並放入 result
        result[i] = callback(arr[i]); 	//呼叫 function 處理原本的 arry
    }   
    return result
}

function double(n){
    return n*2
}
console.log(fakemap(arr,double));
```

### [String.prototype.repeat()](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/repeat)

簡單的重複字串 function ，直接看案例

```js
'abc'.repeat(3) //輸出 abcabcabc
```

直接用迴圈製作`MyRepeat(str,n)`，`str`帶入要重複的字串，`n` 帶入重複次數

```js
function MyRepeat(str,n){
    let result = '';
    for(let i=0;i<n;i++){
        result+=str;
    }
    return result
}
console.log(MyRepeat('abc',3))
```

### [Array.prototype.IndexOf()](https://developer.mozilla.org/zh-TW/docs/Web/JavaScript/Reference/Global_Objects/Array/lastIndexOf)

> `lastIndexOf()` 方法會回傳給定元素於陣列中最後一個被找到之索引，若不存在於陣列中則回傳 -1。搜尋的方向為由陣列尾部向後（即向前）尋找，啟始於 `fromIndex`。

翻成白話的意思：

```js
arr=[1,2,3,1] //假設arr等於這個陣列

console.log(arr.indexOf(4)) //則會在陣列中尋找 4 這個 value，如果 4 不存在則返回 -1

console.log(arr.indexOf(2)) //則會返回 1 (從前方數來的第幾個 index)

console.log(arr.indexOf(3)) //則輸出2

console.log(arr.indexOf(1)) //如果傳入數值為重複的值，則會優先返回第一個找到的 index，所以輸出 0
```

`lastIndexOf()` 就是從 array 後面找回來，功能與 `IndexOf()` 一致

直接實做

```js
arr=[1,2,3,1]
function lastIndexOf(arr,target){
    for(let i=arr.length-1 ; i>=0 ; i--){		//比對成功就返回 i 
        if(arr[i]===target) return i
    }
    return -1  								//記得判斷不存在的值
}
console.log(lastIndexOf(arr,3))
```

### [String.prototype.padEnd() ](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/padEnd)

```js
function padEnd(str, targetLength, padString) {
  let r = str;
  let n = targetLength-str.length;
  if(targetLength<=str.length){
    return r
  }
  for(let i=0;i<n;i++){
    if(i<padString.length){
      r+=padString[i]
    }

      n=n%padString.length
      i=-1;
    }
  }
  return r
```
