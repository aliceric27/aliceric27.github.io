## **日期與時間的創建**

在 C# 中，我們通過 `DateTime` 類型來處理日期與時間。`DateTime` 是定義在 `System` 命名空間下的一個結構。

### **基本創建方式**

- 使用構造函數：

```csharp
DateTime dateTime = new DateTime();
```

這種方式可以創建一個 `DateTime` 的實例，其值預設為 0001-01-01 00:00:00。

### **指定日期與時間**

- 通過具體的年、月、日來創建：

```csharp
DateTime specificDate = new DateTime(2024, 3, 6);
```

這將創建一個表示 2024 年 3 月 6 日的 `DateTime` 實例。

## **日期與時間的操作**

處理日期與時間不僅僅是創建 `DateTime` 實例那麼簡單，更多時候我們需要對它進行各種操作，比如加減日期、格式化輸出等。

### **加減日期**

- 加上一定的天數：

```csharp
DateTime tomorrow = DateTime.Now.AddDays(1);
```

這將獲得明天的日期。

### **格式化日期與時間**

- 將 `DateTime` 實例轉換成特定格式的字符串：

```csharp
string formattedDate = DateTime.Now.ToString("yyyy-MM-dd HH:mm:ss");
```

這將把當前的日期與時間格式化為 "年-月-日 時:分:秒" 的格式。

# C# 中 `TimeSpan` 的使用教學

本篇將介紹另一個類型：`TimeSpan`，它代表時間長度。

## **創建 `TimeSpan`**

`TimeSpan` 提供了多種創建方式，下面是一些基本方法：

### 使用新建操作符

```csharp
TimeSpan timeSpan = new TimeSpan();
```

這種方法將創建一個表示時間長度的 `TimeSpan` 對象，預設值為 0。

### 指定時間

你可以通過提供天、小時、分、秒等來創建一個具體的時間跨度：

```csharp
TimeSpan specificTimeSpan = new TimeSpan(1, 2, 30, 45);
```

這將創建一個代表 1 天 2 小時 30 分 45 秒的時間跨度。

## **`TimeSpan` 的應用**

`TimeSpan` 常用於日期和時間的計算中，比如：

### 加上一段時間

```csharp
DateTime futureDate = DateTime.Now + new TimeSpan(5, 0, 0, 0);
```

這將計算出從現在起 5 天後的日期。

### 時間間隔的計算

```csharp
TimeSpan interval = specificTimeSpan - new TimeSpan(1, 0, 0, 0);
```

這將從上面創建的 `specificTimeSpan` 減去 1 天，計算出新的時間間隔。
