Mac上實用的工具

發表於 2024-04-28 分類於 工具

紀錄一些開發必備的工具，有些工具真的好用到用回 Win 系統就不會用了

Raycast

這個應用不用說了，Google一下就可以查到非常多推薦的常用小工具這邊舉例幾個優點

  1. 快捷鍵快速呼叫

  2. 剪貼簿，和 Windows 比較根本是大砲比雞腿

不僅有歷史剪貼簿功能，也會將所有複製過的東西都分門別類，像是 Hex 色碼、圖片、文字類型…等。

還有快速轉換匯率、快速翻譯、Format Json..如果沒有自己要的，自己刻一個功能也可以

這邊強烈推薦，蘋果護城河又向外延伸了不只一點。

截圖工具

這邊推薦兩個

  1. iShot

    • 圓角截圖

    • 截圖後HightLight

  2. Snipaste

    這款 Windows 上也有，Mac 版本目前沒有付費版、Windows 付費版功能更完整且強大

    不過也是可以加減用、沒有複雜功能也可以截圖後釘選和編輯，感覺吃的效能比 ishot 少一點

Rectangle

開源的視窗快速調整神器，可以快速將視窗定位大小和移動視窗，真的很爽，Mac系統必裝。

BetterDisplay

這款軟體主要對外接螢幕優化的，功能很多像是調整解析度、調整亮度、快速檔案切換、HiDPi 解析度解放

很早買斷 15$ ，現在看竟然還漲價，還有一直更新，╰(⊙Д⊙)╮佛心公司╭(⊙Д⊙)╯

WPS office

Mac 上屌打 MS office 的軟體真的很好用，我用的功能不多，光是可以像 chrome 那樣多開 xls 標籤直接切換就贏了

不過這款稍微吃記憶體，可以斟酌一下。

Arc Browser

這款瀏覽器是 Chrome 基底調教出來的，小功能滿多，現在開發也用習慣了，其實算滿好用的

Mos

外接滑鼠必備，不然滾輪像智障一樣，根本用不了。

KeepingYouAwake

防止電腦睡眠，遠端必備。

Command X

讓 Mac 支援剪下功能…竟然要靠第三方才有這功能。

G Hub

羅技滑鼠用的，可以寫 Lua 腳本實現用側鍵達成 control + ➡️ 切換的功能，改天再寫一篇教學
