LIOJ | 印出金字塔

教學影片解題是使用兩個 function ，一個為輔助 function，專門處理重複的字串

我的解題思路是

  1. 先使用 for 迴圈印出列數

  2. 將金字塔分割為輸出空白 與輸出星星*

  3. 使用 function 帶入需要的規則參數

  4. 將空白用-先替代並照規則輸出

  5. 將星星*輸出

  6. 最後將兩個字串相加就是答案了

九九乘法表 & LIOJ | NM 乘法表

解題思路：

  1. 先將被乘數與乘數分離開來

  2. 先輸出被乘數的迴圈與結果

  3. 因為要完整執行完一次(乘數*被乘數)，所以在最外層加上一個乘數迴圈(跑幾輪乘數)

  4. 最後在跑完迴圈可以輸出分隔線來分隔一個已經完成運算的表

分離乘數、被乘數的優點：

如果考官面試要輸出 13 * 13 甚至 20 * 20 的乘法表都可以生成出來

判斷1~100平方數

平方數的定義：數字開根號後是否為正整數

1 * 1 = 1 、2 * 2 = 2、3 * 3 = 9 …以此類推

第一種方法：將所有數開根號並相乘，看結果是否等同傳進來的數值，是的話回傳 true

第二種方法：直接找出所有平方數

😱LIOJ | 印出聖誕樹

解題思路：拆開來做

  1. 和 LIOJ | 印出金字塔 相同解法 拆開空白與星星

  2. 判斷星星迴圈印完的那一輪加上樹幹部分

  3. 先處理樹幹的邏輯產生數量為 i-1 個樹幹

  4. 處裡樹幹空白的部分產生空白的邏輯為 b-1 個空白

  5. 因為空白的數量都是一致的所以將樹幹與空白的字串相加後放在星星輸出完成後輸出樹幹即為正確答案

這題學到了 .tofix(n)內建函數，為取小數到後面幾位

解題思路：

  1. 寫一個 for 迴圈來查看要執行幾次
  2. 算出總共要跑幾次並帶入第 1 個迴圈
  3. 再寫一個 for 迴圈來將每四組數傳到陣列中並計算印出
  4. 再重複執行n次直到跑完所有 4 個為 1 組的計算
