Array.prototype.map()

map() 方法會建立一個新的陣列，其內容為原陣列的每一個元素經由回呼函式運算後所回傳的結果之集合。

所以 map() 可以帶入 function 並回傳一個新的陣列

Array.prototype.IndexOf()

lastIndexOf() 方法會回傳給定元素於陣列中最後一個被找到之索引，若不存在於陣列中則回傳 -1。搜尋的方向為由陣列尾部向後（即向前）尋找，啟始於 fromIndex。

翻成白話的意思：

lastIndexOf() 就是從 array 後面找回來，功能與 IndexOf() 一致

Array.prototype.reverse()

reverse()方法會原地（in place）反轉（reverses）一個陣列。陣列中的第一個元素變為最後一個，而最後一個元素則變成第一個。

簡單的陣列反轉，直接實作。

indexOf() 方法會回傳給定元素於陣列中第一個被找到之索引，若不存在於陣列中則回傳 -1。

Array.prototype.fill()

fill() 方法會將陣列中索引的第一個到最後一個的每個位置全部填入一個靜態的值。

Array.prototype.join()

join() 方法會將陣列（或一個類陣列（array-like）物件）中所有的元素連接、合併成一個字串，並回傳此字串。

String.prototype.trim()

trim() 方法會移除字串起始及結尾處的空白字元。 本文中的空白字元指所有空格字元（如：空格、欄標、無間斷空格等等）及換行字元（如：換行、回車等等）。

學習到設定 flag 設定

有點像是 開關 (Trigger)，主要邏輯是：

假設一個變數 a=+++123+34+++

我想要將a處理成123+34+++，則要寫一個迴圈判斷，然後再加上開關，因為我不想過濾數字後面的+

所以需要在迴圈碰到 a[i] !== + 的時候將開關設為 true ，表示之後該迴圈後的數都要留著，然後在 if 判斷加上 || 邏輯則會將後面的數值都保留下來
