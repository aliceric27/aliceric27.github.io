Array.prototype.map()

map() 方法會建立一個新的陣列，其內容為原陣列的每一個元素經由回呼函式運算後所回傳的結果之集合。

所以 map() 可以帶入 function 並回傳一個新的陣列

Array.prototype.IndexOf()

lastIndexOf() 方法會回傳給定元素於陣列中最後一個被找到之索引，若不存在於陣列中則回傳 -1。搜尋的方向為由陣列尾部向後（即向前）尋找，啟始於 fromIndex。

翻成白話的意思：

假設 arr=[1,2,3,1]，console.log(arr.indexOf(4)) 則會在陣列中尋找 4 這個 value，如果 4 不存在則返回 -1

console.log(arr.indexOf(2)) 則會返回 1 (從前方數來的第幾個 index )，console.log(arr.indexOf(3)) 則輸出2

反之如果找重複的值 console.log(arr.indexOf(1)) 則會優先返回第一個找到的 index ，所以輸出 0

lastIndexOf() 就是從 array 後面找回來，功能與 **IndexOf()**一致
