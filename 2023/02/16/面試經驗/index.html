設計公司後端

面試辦公室漂亮，主管人不錯，進公司就有狗狗跑出來迎接

提問

  • 上一份工作的詳細工作內容

  • 詢問作品使用到的相關技術方面問題

  • 請我直接修改專案的內容，使購物車的數量限制一個最大數量

    （未成功做出，思路：藉由修改 PHP 使更新時其最大數量被限制）

  • 請我直接在頁面上使用 JQuery 抓 HTML DOM 並將內容放至陣列
    （使用原生 JS 有成功抓到並放入陣列）

  • 請我寫出一個 function (5) 其輸出答案為 1,1,2,3,5 第三位數為前兩位數的總合
    （未成功寫出，卡非常久）
