遊戲後台前端面試經驗

發表於 2023-02-24 更新於 2024-04-27 分類於 Others

遊戲後台前端

面試了一家遊戲公司，聊得很開心。在面試過程中氛圍非常融洽，主管和團隊創辦人都很友好。沒有詢問太多技術問題，而是更關心我的個人背景和經驗。

會後結束 HR 就致電約二次面試主要與技術主管會面

公司提問

  • 職位為開發後台前端頁面 (猜測為串接 API 讓使用者輸入數值，並呼叫對應 API 達到對應修改)

*　職位是否符合期待? (是，我本身對於前端純設計上較不上手，喜歡後端邏輯開發)

  • 公司使用的技術? (Vue)

  • 詢問我關於上份工作的內容(如實回答)

我的提問

  • 為什麼會有這份職缺?（團隊擴編）
  • 對於新人教育是如何?（會讓我給一位資深工程師帶）
  • 對於員工在職培訓或是否提供其他學習管道（每個月有購書津貼可以購買需要的書籍、如果團隊有需要也可以在職培訓）
  • 會隨時變換技術跳轉框架嗎?（不會，主要以穩定為主）
  • 關於工作上專案上會隨時輪調到其他專案嗎?

（主要會依照工作內容來給定，如果說其他專案也是同樣的後台功能，那會統一由同一人負責）

  • 關於工作上專案主要開發人數?

（會依照當前專案進度配置人力）

結論

主要為新創公司的緣故，所以組織較為扁平化，比較沒有升遷機會，相對的給予的福利非常好，彈性上下班、居家辦公、員工還有一個禮拜暑假！？

缺點就是代工產品一樣就是博弈相關的東西，不過如果團隊也願意給新人一個機會我確實會滿想試試看的，畢竟福利跟一般公司真的差太多了😢

最後詢問我預期薪資，當下我只開36K，因為我認為這間公司給薪應該是不會吝嗇大不了再跳其他間，如果能夠勝任再來敘薪我認為是比較合理的，加上我其實也滿想去試試看，與 HR 預約技術主管二面，到時候會再更新。
