Typescript 聲明方式

發表於 2023-03-09 分類於 Javascript ， Typescript

Typescript 常用的聲明方式，因為工作專案使用所以學習，學完後可以看懂8成專案內容了!

常用類型整理

類型 範例 描述
number 1, -33, 2.5 任意數字
string ‘hi’, “hi”, `hi` 任意字符
boolean true、false 布林值true或false
字面常數 其本身 限制變數的值就是該字面常數的值
any * 任意類型
unknown * 類型安全的any
void 空值（undefined） 沒有值（或undefined）
never 沒有值 不能是任何值
object {name:”黃胖胖”} 任意的JS物件
array [1,2,3] 任意JS陣列
tuple [4,5] 元素，TS新增類型，固定長度數組
enum enum{A, B} 枚舉，TS中新增類型

聲明變數

TS 基本的聲明變數方式：

  • 在變數後加上冒號 let a:number即為將 a 聲明為 number，無法賦值 number 外的數值
  • 函數聲明

額外的類型補充

  • 字面常數(字面量)：類似 const 賦值，當指定常數後，變數則無法賦值指定常數外的值

  • any 表示任意值，盡量少用這個類型

  • unknow 為 安全版 any

  • as 斷言類型，強制告訴解析器使用什麼類型來對變數解析
    語法：變數 as 類型、<類型>變數
  • void 與 never
    void 表示為回傳空值，never 表示永不回傳值，通常用於除錯

物件相關類型

  • 物件相關類型宣告方式：
  1. 通常使用{}來指定物件可以包含哪些屬性
  2. 語法：{屬姓名:屬性值,屬性名:屬性值}
  3. 在屬性名後面加上?，表示屬性可選(賦值時該屬性可有可無)，沒加則為必需值
  • 設置結構函數的類型聲明：
    強制該變數需要按照特定規則傳入指定參數與指定類型，返回值也要按照設定返回

使用語法：(參數:類型,參數:類型...)=>返回值(類型)

  • 陣列的類型聲明
    使用語法：類型[]、Array<類型>
  • 元組的類型聲明
    元組表示固定長度的陣列
    語法：[類型,類型,類型]
  • enum 列舉(枚舉)類型
    將所有可能情況預先設定好，並直接使用
  • & 表示與(同時)
  • 類型的別名
    type 自訂變數 = 聲明條件
    使用如果聲明一個條件判斷且要複用可以使用此宣告方式
