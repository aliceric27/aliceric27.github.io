發表於 2023-03-07 更新於 2024-03-06 分類於 Javascript

來複習一下最基本的東西 Javascript ES6 相關語法~

變數宣告

分辨 var、let、const差異?

var：如果使用 var會導致變數洩漏到全域，導致域名汙染。

let：如果使用 let則作用域限定為此 代碼塊區域 {} 內。

const：無法被重新賦值。

箭頭函數

  1. 寫法為 () => {}
  2. 參數只有一個時，可以省略小括號 ()
  3. 省略大括號 {} 的話，代表直接 return 第一行程式碼
  4. 如果返回值或參數有包含 {} ，需要在外圍加上括號 ()
  5. 如果有多行程式，需要加上大括號 {}

覺得第4點滿重要，因為使用 Vue 框架時常需要返回物件，會忘記加 ()

剩餘參數

特性

  • 取得剩下的參數，並存為一個陣列。
  • 剩餘參數在傳入時，必定是位於最後一位，並且只能有一個剩餘參數。
  • 剩餘參數的值在沒有傳入實際值時，會變為一個空陣列，而不是 undefined。
  • 又叫做「剩餘運算子」「其餘參數」「其餘運算子」
  • ESLint（airbnb）規則：不要使用 arguments 語法，使用 rest 語法 ... 替代
