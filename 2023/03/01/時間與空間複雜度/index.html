LIOJ | 簡易排序

使用到 .sort內建函數

簡單來說.sort可以傳入一個 function 並依照 return 的值對原本陣列進行排序

簡易記法，假設傳入的值為 a,b 分別是 3 與 5

如果 return 值為 -2 (a-b) 則陣列保持不變(由小到大)

如果 return 值 2 (b-a) 則a,b對調，且陣列排序(由大到小)

使用counting sort排序方式

原理：因為數入數值是固定的，所以先準備好特定範圍內的容器，然後分別放入

最後再用迴圈讀取每個容器數值為多少就是需要重複印出幾次
